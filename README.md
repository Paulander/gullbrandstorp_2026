# Gullbrandstorp Ingenjörsbyrå AB

Modern bilingual landing page for Gullbrandstorp Ingenjörsbyrå AB.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

Deploy on Vercel as a Vite project. Build command: `npm run build`. Output directory: `dist`.
